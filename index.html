Projects

cpartlo

This is a HTML5/CSS3 fuild design built on PHP and Apache. The website is designed around a fluid website layout all widths as percentages and font-size as em. The website is controlled layout is controlled through three different CSS files using CSS media queries for screen, handheld, and print. From the screen and handheld CSS files these are broken down further into large and small sizes for each of their respective type.

Fantasy Football Analytics in R

Fantasy Football Analytics in R is a blog for harnessing the power of statistics, data analysis, and R to improve your performance in fantasy football. This GitHub repository includes R scripts and data files for conducting the analyses in R as described on the blog.

Flight Plan Visualization

Flight Plan Visualization is a web based application currently in development to parse flight plans for some of the major airlines with the aim of exporting the flight plan data into multiple different formats which are widely available across multiple devices.

udportfolio

udportfolio is focused on website performance optimization through the critical rendering path. This is a test bed for optimizing the critical rendering path through proper construction of the document object model of HTML, and CSS. Additionally, understanding tree construction, HTTP caching, PageSpeed, image optimization, eliminating unnecessary download and interacting with JavaScript.
